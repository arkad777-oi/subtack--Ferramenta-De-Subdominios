# subtack--Ferramenta-De-Subdominios

Este projeto em Python permite:

Descobrir subdomínios ativos (`-s`)

Rodar portscan com **nmap** (`-p`)

Verificar possíveis **takeovers** (`-t`)

Crie o ambiente virtual automaticamente:

python3 meu.py

depois execute com "source meuambiente/bin/activate

e agora pode rodar normalmente
